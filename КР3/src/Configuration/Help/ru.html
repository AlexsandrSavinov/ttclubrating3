Общие кнопки управления

 Создать (Добавить (Insert)) - добавляет документ, элемент справочника или строку.

Изменить(F2) - открывает форму для изменения элемента, или позволяет изменить строку если это возможно.

Удалить - помечает элемент справочника или документ пометкой на удаление, это значит что из базы он не удаляется совсем, но обычно такие элементы потом не используют.Так же удаляет строки в таблицах.

Удалить непосредственно - полностью удаляет элемент из базы (не рекомендуется использовать).

Сортировка - по возрастанию и убыванию, действует практически на любые колонки списков и таблиц.

Перемещение - строк в заданных направлениях.

Справка (F1) - доступна во всех формах программы по нажатию горячей клавиши, а так же где то расположена прямо на форме.

Кнопка выбора (F4)- кнопка с 3 точками, позволяет выбрать соответствующее значение для поля ввода.

Кнопка открытия - открывает форму элемента справочника для просмотра или изменения, из поля ввода.

Кнопка очистки - очищает поле ввода от введенного значения.
