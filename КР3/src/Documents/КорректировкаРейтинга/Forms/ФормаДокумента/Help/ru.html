Если у участника уже есть рейтинг, его можно начислить с помощью этого документа или указать его рейтинг при проведении документа соревнования. В табличную часть вводятся данные, для записи начального рейтинга (не обязательно вводить все данные).
   Документ автоматически может создаться из обработки загрузки участников.
