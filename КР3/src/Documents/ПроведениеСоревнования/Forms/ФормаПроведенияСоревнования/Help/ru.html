Работа с групповым этапом

 Для быстрого ввода партий в основной таблице:

Что бы, быстро ввести сыгранную игру необходимо:

  1. Выделить ячейку игры, выигравшего участника.
  2. Ввести количество проигранных партий.
    1. 0 - не выиграл ничего.
    2. 1 - выиграл один сет.
    3. 2 - выиграл два сета.
    4. 3 - выиграл три сета.
    5. Т (в русской раскладке,регистр не имеет значения) - значит сопернику устанавливается техническо поражение.
